self-practice ใน week นี้เป็นการเรียนซ้ำเรื่อง Fetch API เนื่องจากไม่เข้าใจ จึงได้ย้อนดูคลิปซ้ำ 
ซึ่งไฟล์ AsyncSync.js และ fetchUtils.js คือไฟล์ที่ self-practice และเขียนเป็น comment ที่เรียนรู้กำกับไว้เพื่อความเข้าใจ

-> AsyncSync.js
- Part 01 เรียนรู้เรื่อง Asynchronous และ Synchronous เพื่อศึกษาความแตกต่างและทำความเข้าใจ เพราะเบื้องต้นเคยได้ยินบ่อยเกี่ยวกับ async แต่ยังไม่ค่อยแน่ใจ
- Part 02 Promises เป็นความรู้ใหม่มาก ซึ่งต้องทำความเข้าใจเพิ่มเติมเยอะ ทั้งนิยามของ Promises ทฤษฎี และศึกษา function async ที่ return promise จริงๆ
- Part 03 เป็น Knowlegde ที่ต้องรู้เกี่ยวกับเรื่องนี้ จึงกำกับเอาไว้
- Part 04 พอรู้ว่า Promises คืออะไร ก็รู้ว่ามันไม่มีการจัดการกับโปรแกรมที่คืนค่าเป็น Promises เลย จึงต้อง handle มันด้วย 2 วิธี 
  ส่วนตัวมองว่า .then().catch() เข้าใจง่ายและเขียนง่ายกว่า แต่ในการ Dev จริง async-await น่าจะเหมาะกว่า
- Part 05 เป็นวิธีการ Fetch API จริงๆ พอได้เรียนรู้ซ้ำทำให้ได้รู้โครงสร้างของ REST APIs และเข้าใจตอนนำไปทำ intproj25 มากขึ้น

-> fetchUtils.js
- Part 06 จากใน project qoute manager อาจารย์สอน fetch โดยใช้ async-await แต่ได้ลองเขียนแบบที่ใช้ .then().catch() เพิ่มไป
